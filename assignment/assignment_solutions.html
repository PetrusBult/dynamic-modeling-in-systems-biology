Derivation steady state values

The ODE system is given by:

$$\frac{dx}{dt} = -mx - d_1(x - c)$$ $$\frac{dy}{dt} = mx - d_2 y$$

At steady state -> both derivatives to 0.

For the tissue compartment ($x_{ss}$):

$$\frac{dx}{dt} = 0 \implies -mx - d_1(x - c) = 0 \implies x_{ss} = \frac{d_1 c}{m + d_1}$$

For the blood compartment ($y_{ss}$):

$$\frac{dy}{dt} = 0 \implies mx - d_2 y = 0 \implies y_{ss} = \frac{m x_{ss}}{d_2}$$

Substituting $x_{ss}$: $$y_{ss} = \frac{m}{d_2} \cdot \frac{d_1 c}{m + d_1}$$

Stability

To check stability, look at Jacobian at the steady state.

Jacobian Matrix:

$$J = \begin{bmatrix} \frac{\partial f_1}{\partial x} & \frac{\partial f_1}{\partial y} \\ \frac{\partial f_2}{\partial x} & \frac{\partial f_2}{\partial y} \end{bmatrix} = \begin{bmatrix} -(m + d_1) & 0 \\ m & -d_2 \end{bmatrix}$$

Eigenvalues:

The eigenvalues are then given by: $$\lambda_1 = -(m + d_1)$$ $$\lambda_2 = -d_2$$

Both eigenvalues are negative (since $m$, $d_1$, and $d_2$ are all positive parameters). This means the steady state is asymptotically stable.

Assignment: implement & combine global search with local search.

Alternatively : sample many starting points & optimize from best x.

(I already had some functionality lying around for this - so I'll do this first). The fit_model method (estimation.jl) is responsible for finding the best parameter values. For this it takes the weighted_sse (can also be found in estimation.jl) between the model simulation and observed data as loss function. It now uses LBFGS as optimization method - but can be easily altered. By finding the parameters that minimize the sse-loss, the 'best' parameters are obtained. The code below first does latin-hypercube sampling of potential starting points that lie between the lower and upper bound. It currently does 10000 initial samples (n_samples) - it then calculates the loss for all of them, picks the best 100 (n_starts) to optimize further (local optimization). The final parameters are the ones minimizing the loss best.

NOTE: the multistart with so many samples is a bit overkill - when I started with local optmization from the 'guesed' starting point I also already found the parameters.

using ForwardDiff, LinearAlgebra

"""
Compute parameter covariance matrix from the Hessian of the loss function.
Returns covariance matrix, standard errors, and correlation matrix.
"""
function compute_covariance(loss_fn, n_obs, param_values)
    # Compute Hessian at the optimum using ForwardDiff module
    hessian = ForwardDiff.hessian(p -> loss_fn(p, nothing), param_values)
    
    # Covariance matrix is inverse of Hessian (for maximum likelihood/least squares)
    # We need to scale by the residual variance
    # For weighted least squares: Cov = H^(-1) * 
    # where   SSE / (n_observations - n_parameters)
    
    n_params = length(param_values)
    
    # Compute residual variance (SSE at optimum)
    sse = loss_fn(param_values, nothing)
    residual_variance = sse / (n_obs - n_params)
    
    # Covariance matrix (handle potential singularity)
    cov_matrix = try
        inv(hessian) * residual_variance
    catch
        @warn "Hessian is singular, using pseudo-inverse"
        pinv(hessian) * residual_variance
    end
    
    # Standard errors (square root of diagonal)
    # Diagonal of covariance matrix contains the variances of each parameter
    std_errors = sqrt.(abs.(diag(cov_matrix)))
    
    # Correlation matrix
    D_inv = Diagonal(1 ./ std_errors)
    corr_matrix = D_inv * cov_matrix * D_inv
    
    return cov_matrix, std_errors, corr_matrix
end
